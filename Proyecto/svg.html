Etiquetas SVG

Etiqueta <circle> (Círculo)

La etiqueta <circle> se utiliza para dibujar círculos en SVG.

Ejemplo de código:

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="blue" />
</svg>

Para más información, visita la documentación de W3S.

Etiqueta <rect> (Rectángulo)

La etiqueta <rect> se usa para dibujar rectángulos.

Ejemplo de código:

<svg width="100" height="100">
    <rect width="80" height="50" x="10" y="25" fill="green" />
</svg>

Resultado en el navegador:

Consulta la documentación de W3S para más detalles.

Etiqueta <line> (Línea)

La etiqueta <line> dibuja una línea recta entre dos puntos.

Ejemplo de código:

<svg width="100" height="100">
    <line x1="10" y1="10" x2="90" y2="90" stroke="red" stroke-width="2" />
</svg>

Resultado en el navegador:

Para más información, visita la documentación de W3S.

Etiqueta <ellipse> (Elipse)

La etiqueta <ellipse> se usa para dibujar elipses.

Ejemplo de código:

<svg width="100" height="100">
    <ellipse cx="50" cy="50" rx="30" ry="20" fill="purple" />
</svg>

Resultado en el navegador:

Revisa la documentación de W3S para aprender más.

Etiqueta <polygon> (Polígono)

La etiqueta <polygon> define formas de múltiples lados a partir de una serie de puntos.

Ejemplo de código:

<svg width="100" height="100">
    <polygon points="50,15 90,80 10,80" fill="orange" />
</svg>

Para más información, visita la documentación de W3S.
