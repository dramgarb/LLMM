Etiquetas HTML

Etiqueta <p> (Párrafo)

La etiqueta <p> se utiliza para definir párrafos en HTML.

Ejemplo de código:

<p>Este es un párrafo de ejemplo.</p>

Resultado en el navegador:

Para más información, visita la documentación de W3S.

Etiqueta <a> (Enlace)

La etiqueta <a> se usa para crear enlaces a otras páginas o recursos.

Ejemplo de código:

<a href="https://www.ejemplo.com">Visitar Ejemplo</a>

Resultado en el navegador:

Para más detalles, revisa la documentación de W3S.

Etiqueta <img> (Imagen)

La etiqueta <img> se usa para incrustar imágenes en una página HTML.

Ejemplo de código:

<img src="ruta/imagen.jpg" alt="Descripción de la imagen">

Resultado en el navegador:

Más información en la documentación de W3S.

Etiquetas <ul> y <li> (Lista)

La etiqueta <ul> define una lista sin orden, y <li> define los elementos de la lista.

Ejemplo de código:

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Resultado en el navegador:

Consulta la documentación de W3S para más detalles.

Etiqueta <table> (Tabla)

La etiqueta <table> se usa para crear tablas en HTML.

Ejemplo de código:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>
</table>

Resultado en el navegador:

Revisa la documentación de W3S para aprender más.
